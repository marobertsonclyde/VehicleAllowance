Design notes

What this is

Static HTML/CSS wireframes. No React, no Dataverse, no real uploads. The goal is to walk the product before a line of Code App TypeScript is written.

Brand

Clyde Companies core palette (Clyde Blue #0033A1 dominant, Pearl Gray backgrounds, Open Sans throughout). No shadows, no gradients — Clyde rules.

Component language

Every UI pattern maps 1:1 to Fluent UI v9: buttons, inputs, cards, data grids, tab lists, message bars, drawers. The real Code App should be able to adopt @fluentui/react-components without a re-skin.

Responsive

Mobile-first. Breakpoints at 600px (tablet) and 1024px (desktop). On mobile: single column, hamburger drawer, bottom nav, vertical wizard, card-per-row in place of tables. On desktop: persistent left rail, 2-column wizard, inline tables.
