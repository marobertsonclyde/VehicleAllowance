# Data Model Reference — Vehicle Allowance Program

All custom tables use the publisher prefix `va_`. All tables use Standard ownership (owned by Business Unit).

## Table Inventory

| Table (Schema Name) | Display Name | Description |
|---|---|---|
| `va_AllowanceApplication` | Allowance Application | Central record per application lifecycle |
| `va_Vehicle` | Vehicle | Vehicle details per application |
| `va_InsurancePolicy` | Insurance Policy | Policy record with expiry tracking |
| `va_Document` | Document | One row per uploaded file |
| `va_DocumentChecklist` | Document Checklist | 4 required items per application |
| `va_AllowanceRecord` | Allowance Record | Living active state once approved |
| `va_CompanyEntity` | Company Entity | 9 legal entities (pre-seeded) |
| `va_EligibleTitle` | Eligible Title | Job titles mapped to allowance levels |
| `va_AllowanceLevelConfig` | Allowance Level Config | Level → MSRP minimum + monthly amount |
| `va_AuditLog` | Audit Log | Immutable event trail |
| `va_Notification` | Notification | Record of all outbound notifications |
| `va_InsuranceReminderSchedule` | Insurance Reminder Schedule | Per-policy reminder tracking |
| `va_ReminderConfig` | Reminder Configuration | Admin-configurable reminder intervals |

## va_AllowanceApplication

**Primary key:** `va_allowanceapplicationid`  
**Name field:** `va_name` (auto-generated: VA-YYYY-NNNNN)

| Column (Schema Name) | Display Name | Type | Notes |
|---|---|---|---|
| `va_name` | Application Number | Text | Auto-generated by Flow 1 |
| `va_applicantid` | Applicant | Lookup → SystemUser | AAD-authenticated employee |
| `va_personnelnumber` | Personnel Number | Text | From Dynamics via Fabric SQL |
| `va_applicationType` | Application Type | Choice | New Opt-In / Vehicle Update / Annual Renewal |
| `va_status` | Status | Choice | Full lifecycle status (see below) |
| `va_allowanceLevel` | Allowance Level | Choice | A / B / C / D |
| `va_monthlyAllowanceAmount` | Monthly Allowance Amount | Currency | From va_AllowanceLevelConfig at approval |
| `va_isElectricVehicle` | Electric Vehicle | Yes/No | Default: No |
| `va_evChargingAllowance` | EV Charging Allowance | Currency | $330 when EV = Yes |
| `va_totalMonthlyAllowance` | Total Monthly Allowance | Currency | Calculated field |
| `va_effectiveDate` | Effective Date | Date | 15th/16th rule applied by Flow 6 |
| `va_submittedOn` | Submitted On | DateTime | Set by Flow 1 |
| `va_eligibilityDeadline` | Eligibility Deadline | Date | 60-day window or Aug 1 |
| `va_companyEntityId` | Company Legal Entity | Lookup → va_CompanyEntity | For endorsement addressing |
| `va_equipmentLeaderReviewDate` | Equipment Leader Review Date | DateTime | |
| `va_equipmentLeaderDecision` | Equipment Leader Decision | Choice | Approved / Returned / Rejected |
| `va_equipmentLeaderNotes` | Equipment Leader Notes | Multiline Text | Hidden from Employee role |
| `va_directorReviewDate` | Director Review Date | DateTime | |
| `va_directorDecision` | Director Decision | Choice | Approved / Returned / Rejected |
| `va_directorNotes` | Director Notes | Multiline Text | Hidden from Employee role |
| `va_presidentApprovalRequired` | President Approval Required | Yes/No | True for New Opt-In |
| `va_presidentApprovalDate` | President Approval Date | DateTime | |
| `va_presidentDecision` | President Decision | Choice | Approved / Rejected |
| `va_payrollNotifiedDate` | Payroll Notified Date | DateTime | |
| `va_aiValidationScore` | AI Validation Score | Decimal | 0–100 composite |
| `va_aiValidationSummary` | AI Validation Summary | Multiline Text | AI-generated summary |
| `va_aiFlaggedIssues` | AI Flagged Issues | Multiline Text | Comma-separated flag codes |
| `va_aiAutoApprovalEligible` | AI Auto-Approval Eligible | Yes/No | Score ≥ threshold AND all flags pass |
| `va_parentApplicationId` | Parent Application | Lookup → va_AllowanceApplication | Links renewals/updates to original |

**Status Choice Values (in order):**
`Draft` → `Submitted` → `AI Review` → `Equipment Leader Review` → `Director Review` → `President Review` → `Payroll Notification` → `Active` | `Rejected` | `Returned to Employee` | `Withdrawn` | `Terminated`

### Policy Type
`Auto` | `Umbrella-Personal Excess`

### Endorsement Type
`Additional Insured` | `Additional Interest`

### Body Type
`Pickup` | `SUV` | `Other`

### AI Processing Status
`Pending` | `Processing` | `Completed` | `Failed`
